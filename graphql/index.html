Giới thiệu về GraphQL

GraphQL là một ngôn ngữ truy vấn cho API của bạn. Sử dụng hệ thống loại mà bạn xác định cho dữ liệu của mình. GraphQL không bị ràng buộc với bất kỳ cơ sở dữ liệu hoặc công cụ lưu trữ cụ thể nào và thay vào đó được hỗ trợ bởi mã và dữ liệu hiện có của bạn.

Dịch vụ GraphQL được tạo ra bằng cách xác định các kiểu (type) và trường (field) trên các kiểu đó, sau đó cung cấp các hàm cho từng trường trên mỗi kiểu. Ví dụ: một dịch vụ GraphQL cho chúng tôi biết người dùng đã đăng nhập là ai cũng như tên của người dùng đó có thể trông giống như sau:
 
type Query {
  me: User
}

type User {
  id: ID
  name: String
}

Cùng với các function cho từng trường trên từng loại:

function Query_me(request) {
  return request.auth.user;
}

function User_name(user) {
  return user.getName();
}

Truy vấn cho với ví dụ trên ta sẽ có code sau:

{
  me {
    name
  }
}

Kết quả:

{
  "me": {
    "name": "Luke Skywalker"
  }
}
